# 만들고 있는 게임: 이상한 던전

**이상한 던전 시리즈** 중 **초코보의 이상한 던전**을 목표로 만들고 있습니다.

캐릭터 및 UI등은 미모지를 활용하였습니다.

AI는 사용하지 말라고 하셨지만, 게임으로서 중요한 부분 중 하나인 오프닝만 제미나이에게 부탁했습니다.

"게임보이 켰을때 화면처럼 나왔으면 좋겠다." 말도 하고, 오프닝송도 마비노기 모바일 MML 코드를 음계로(제미나이가...) 변환시키고, 그걸 다시 콘솔 코드로(제미나이가...) 변환시키는 작업을 진행했습니다.

그 과정에서 제미나이를 조교해 지금의 오프닝을 만들었습니다.

브금 끝날때쯤 플리즈 애니키 뜨고, 키 누르면 다음 화면 넘어가게 해달라...
